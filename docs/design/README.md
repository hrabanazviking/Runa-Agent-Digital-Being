# docs/design/

Long-form design explorations. Things that informed architecture but are not themselves the spec.

This is where bigger conceptual writing goes — idea harvests from previous projects, deep dives into specific subsystems, sketches of features not yet committed to.

## Imported plundered material

- `RUNA_ADVANCED_AGI_ENGINEERING_GUIDE.md` — deep-engineering reference, large.
- `WORLD_MODELING_SKILL.md` — world-model concept (also lives in NSE / MindSpark / others).
- `RUNA_ECOSYSTEM_IDEA_HARVEST.md` — captured ideas across the ecosystem before they had a home.
- `RUNAFREYJASDOTTIR_GITHUB_CODE_HARVEST.md` — survey of Volmarr's prior code that may seed this project.

When any of these matures into an actual architectural commitment, the commitment is written to `docs/architecture/` and an ADR is added to `docs/decisions/`. The source material stays here as the design lineage.
