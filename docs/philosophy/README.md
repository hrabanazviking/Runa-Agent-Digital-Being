# docs/philosophy/

The worldview that gives Runa her shape. Heathen, Mythic, modern-Viking, techno-spiritual — the ethos sits underneath every architectural choice and is not optional reading for contributors who want to understand why the code looks the way it looks.

The compact statement of this philosophy lives at `PHILOSOPHY.md` in the repo root. The long-form sources live here.

## Imported material

- `Heathen_Third_Path_and_Cyber-Viking_Ethos.md` — the spiritual and political framing of the ecosystem.
- `Volmarr_writings_philosophy.md` — Volmarr's collected philosophical writings that the project's worldview draws from.
- `The_Saga_of_Runa.md` — Runa's mythic origin and character narrative.

These are read alongside `PHILOSOPHY.md` (root) and `RULES.AI.md` (root) to understand the operating ethos.
