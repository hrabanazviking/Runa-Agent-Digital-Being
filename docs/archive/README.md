# docs/archive/

Retired documents. Nothing here is current; everything here is preserved.

## Rules

- Documents are moved here when their replacement lands. The replacement document links back to its predecessor here.
- Nothing is ever deleted from this folder. If a document is wrong, it gets a dated note at the top saying so, but the original wording stays.
- Filenames carry the original name plus an ISO date suffix: `<original-name>--retired-2026-05-17.md`.
- This folder is *not* a place to dump uncertain material. Uncertain material lives in `docs/design/` until it earns archival or canonical status.
