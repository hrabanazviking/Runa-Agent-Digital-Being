# examples/configs/

Fully-worked example configurations. Each is a complete `~/.runa/config/runa.yaml` for a specific deployment shape, suitable to copy-and-edit.

Planned examples:
- `pi-only.yaml` — single Pi 5, local Ollama, no chat bridges.
- `pi-with-discord.yaml` — single Pi 5 with Discord bridge.
- `dev-laptop.yaml` — local dev on a laptop, with hot-reload and verbose logging.
